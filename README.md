# SQL Learning Hub — Interactive SQL Learning Platform

## 📌 Overview
This project is a **Database Management System (DBMS)-based web application** developed as part of coursework.

The main objective is to:
- Implement core DBMS concepts
- Provide an **interactive platform** for learning SQL
- Help students understand SQL through **simple visualizations**

The application allows users to interact with a database using a web interface and perform operations like storing, retrieving, and managing data efficiently.

## 🚀 Features
- User-friendly web interface  
- Database integration for storing and retrieving data  
- CRUD operations (Create, Read, Update, Delete)  
- Structured schema design using DBMS concepts  
- Real-time interaction with backend  

---

## 🛠️ Technologies Used
- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Flask  
- **Database:** SQLite / SQLAlchemy  
- **Hosting:** Render  

---

## 🗄️ Database Design
- Tables designed based on project requirements  
- Relationships maintained using primary and foreign keys  
- Normalization applied to reduce redundancy

## 🧠 Key Concepts Used
- Relational Database Design  
- SQL Queries  
- Normalization  
- Primary and Foreign Keys  
- Data Integrity
